## Your Three-Tier Study Focus

### MASTER THESE FIRST (Tier 1):
1. Direct Proof - how to start from hypothesis, work to conclusion
2. Contrapositive vs. Converse - the most confused distinction
3. Mathematical Induction - THE tool for "for all n" statements
4. Strong Induction - when you need multiple prior cases
5. Proof by Contradiction - for "impossible" claims
6. Set Proofs - A ⊆ B, A = B structure
7. Biconditional Proofs - "if and only if" = both directions
8. Core Definitions - theorem, lemma, corollary, definition

**Why?: These appear in almost every exam. Master = confidence.**

### THEN LEARN (Tier 2):
1. Proof by Cases - for even/odd, positive/negative, etc.
2. Recursive Functions - how recursion and induction fit
3. Existence Proofs - finding a witness
4. Proof Selection - the flowchart thinking

### FINALLY (Tier 3):
1. Uniqueness Proofs - beyond existence
2. Equivalence proofs - chaining equivalences
3. Divisibility proofs - specific techniques

## Pro Tips from the Guide

1. **Definitions first** - Always write out definitions at the start of your proof. This transforms the problem.

2. **State what you're proving** - In induction, explicitly write P(n). In proofs, state the claim. Don't just work through algebra.

3. **Use your assumption** - Every hypothesis must be used. If you proved something without using the hypothesis, your proof is wrong.

4. **Check your base case** - For induction, make sure the base case actually holds. Test it!

5. **Cover all cases** - Proof by cases must be exhaustive. Missing one case = invalid proof.

6. **Memorize templates** - Don't improvise. Use the exact sentence structures given. They're tested and clear.

7. **Practice actively** - Don't just read. Do problems. Write out proofs. Test yourself.

8. **Partial credit is real** - If you get stuck, write what you KNOW. Partial credit is better than blank.
