## Course Design

### Structure

- **30 classes** across 9 modules (3 classes per module)
- **9 labs** — hands-on implementation with starter code and solution keys
- **9 assignments** — proof + code assessments; even-numbered modules include integrative assignments spanning prior modules
- **Module 9 capstone** — synthesizes the entire course

### Each Class Contains

1. **Learning Objectives** — what students will be able to do
2. **Concept Overview** — clear exposition of each topic
3. **Formal Notation** — standard mathematical symbols and definitions
4. **Worked Examples** (3–5) — escalating difficulty, bridging math and code
5. **Proof Techniques Spotlight** — the proof method(s) most relevant to the topic
6. **Java Deep Dive** — complete, runnable Java code with full methods
7. **Historical Context** — brief history of each concept
8. **Slide-Ready Problem Sets** (A/B/C) — conceptual, proof, and Java sets of 3–4 problems each
9. **Full Problem Set** — DM (5–7), Java (5–7), Bridge (2–3) problems
10. **Complete Solutions** — for all problems

### Proof Techniques Progression

| Module | Techniques Introduced | Applied To |
|--------|-----------------------|------------|
| 1 | Direct proof, counterexample | Parity, divisibility |
| 2 | Existence, uniqueness | QRT, FTA |
| 3 | Contrapositive, contradiction, ∀/∃ | Logic, programs |
| 4 | Element-chasing | Set identities |
| 5 | Function composition proofs | Inject/surject/biject |
| 6 | Weak, strong, structural induction | Sums, recursion, trees |
| 7 | Counting arguments, pigeonhole | Combinatorics |
| 8 | Loop invariants, Hoare logic, termination | Algorithm correctness |
| 9 | All of the above synthesized | Trees, heaps, BSTs, complexity |
