Technical Notes
🌍 Geoblock — Canada is Fine
Canada is not on Polymarket's geoblock list — APIs work directly from here. The POLY_PROXY env var is still available in the local server as a useful tool: set it to socks5://127.0.0.1:1080 to route Polymarket calls through any VPN or proxy (useful for testing, region simulation, or corporate networks). Binance has no restrictions. The online GitHub Pages version still routes through corsproxy.io as a convenience.
📡 Binance WebSocket URL
wss://stream.binance.com:9443/ws/btcusdt@kline_1m — 1-minute BTC/USDT klines. Each message: { k: { t, o, h, l, c, v, x } } where x = candle closed. The local server relays this to browser clients at ws://localhost:3000/ws/btc. This removes the need for each browser tab to open its own upstream connection.
🔑 Maker Rebates — Fee Formula
fee_equivalent = C × feeRate × p × (1 - p) where C = shares, p = price, feeRate = 0.072 (Crypto). Daily rebate: (your_fee_eq / total_fee_eq) × rebate_pool. Rebate pool = 20% of taker fees collected in that market. Makers NEVER pay fees — posting limit orders (GTC) is free. Only takers pay.
🔗 Key API Endpoints
gamma-api.polymarket.com/markets?search=btc-updown-5m&active=true&order=endDate — BTC markets
clob.polymarket.com/books?token_id={tokenId} — order book
clob.polymarket.com/prices-history?market={conditionId}&interval=1d&fidelity=1 — price history
data-api.polymarket.com/trades?market={conditionId} — trade history (research needed)
api.binance.com/api/v3/klines?symbol=BTCUSDT&interval=1m&limit=50 — REST candle fallback
